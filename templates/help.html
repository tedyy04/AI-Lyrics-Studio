Setup & Installation

Before starting, ensure you have the following components installed on your system.

Python 3.8+

Required runtime for the backend logic.

FFmpeg (Critical)

Essential for all audio processing tasks.

macOS: brew install ffmpeg
Linux: sudo apt install ffmpeg
Win: Download & Add to PATH

1. Clone the repository:

git clone https://github.com/tedyy04/AI-Lyrics-Studio.git
cd AI-Lyrics-Studio

2. Create Virtual Environment:

python3 -m venv venv
source venv/bin/activate

3. Install Dependencies:

pip install fastapi uvicorn python-multipart jinja2 torch openai-whisper demucs pydub aiofiles soundfile

*Note: 'soundfile' is required to fix backend I/O errors on macOS.

Launch the FastAPI Server

uvicorn main:app --reload --port 8000

Open Browser:

http://localhost:8000

Help page loads first. Click "Launch Studio" to enter.

Troubleshooting

üîç

FileNotFoundError: ... 'ffmpeg'

System cannot find FFmpeg executable.

Fix: Install FFmpeg and add to system PATH.

RuntimeError: Couldn't find appropriate backend

Torchaudio missing I/O library.

Fix: pip install soundfile

Demucs exited with status 1

Environment issue or missing dependency.

Fix: Ensure command is run inside activated 'venv'.
