Project Overview

The AI Audio Studio is a full-stack web application designed to bridge the gap between raw audio and usable content. By leveraging state-of-the-art AI models, it offers two distinct processing modes: Song Mode for vocal isolation and lyrics generation, and Podcast Mode for high-fidelity speech transcription. This interactive documentation explores the architecture, performance, and usage of the system.

Core Capabilities

Vocal Isolation

Powered by Demucs. Separates vocals from background music with high precision.

Detail: Converts audio to 16kHz WAV, runs Demucs source separation, and extracts the vocal track for further processing.

Smart Transcription

Powered by OpenAI Whisper. Generates subtitles (SRT, VTT, LRC) and raw text.

Detail: Supports multiple model sizes (Base to Large-v3). Automatically generates synchronized lyrics with timestamps.

Liquid Glass UI

A modern, immersive interface with fluid animations and glassmorphism.

Detail: Built with TailwindCSS and custom CSS animations. Features a custom audio player with seek support and synchronized lyric highlighting.

          Troubleshooting Center
        

        

          
        


        

          
          

            

              FileNotFoundError: [Errno 2] No such file or directory: 'ffmpeg'
            

            

              System cannot find the FFmpeg executable.
            

            

              Fix: Install FFmpeg via brew or apt and ensure it's in your PATH.
            

          

          
          

            

              RuntimeError: Couldn't find appropriate backend
            

            

              Torchaudio is missing I/O libraries for audio files.
            

            

              Fix: pip install soundfile
            

          

          
          

            

              Demucs exited with status 1
            

            

              Python environment issue or missing dependency.
            

            

              Fix: Check you are using the venv python interpreter.
